# Uninstallation Guide

## Standard Uninstall (Recommended)

1) Odoo → Apps
2) Search “AutoInfo Sale Order Invoice Amount”
3) Click Uninstall

## What Changes After Uninstall

- Additional fields added by this module are removed from UI and database registry.
- Views added by this module are removed.
- The base module `sale_order_invoice_amount` remains untouched.

## Temporary Disable (Without Uninstall)

- Go to Apps → Find the module → Click “Deactivate” (if available in your Odoo build)
- Or remove the module folder from addons_path and restart Odoo (not recommended for production without change control)
